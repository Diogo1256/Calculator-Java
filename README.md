# 🧮 Calculadora Java

Uma calculadora simples desenvolvida em Java para realizar operações matemáticas básicas através do terminal.

## 🚀 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 📥 Entrada de dados pelo usuário
- 📤 Exibição do resultado em tempo real

## 📚 Conceitos praticados

- Variáveis e tipos de dados
- Operadores aritméticos
- Entrada e saída de dados
- Estruturas condicionais
- Lógica de programação
- Classe `Scanner`

## 💻 Tecnologias

- Java
- IntelliJ IDEA

## ▶️ Exemplo de uso

```text
Digite o primeiro número: 10
Digite o segundo número: 5

Escolha uma operação:
1 - Soma
2 - Subtração
3 - Multiplicação
4 - Divisão

Opção: 3

Resultado: 50
